# Omics Workspace Inspection

Codex skill for inspecting omics project workspaces, reconstructing input-output provenance, estimating analysis progress, and reporting reproducibility requirements.

## Use

Install by copying `SKILL.md` into a Codex skill directory, for example:

```bash
mkdir -p ~/.codex/skills/omics-workspace-inspection
cp SKILL.md ~/.codex/skills/omics-workspace-inspection/SKILL.md
```

The skill is intended for bioinformatics, sequencing, spatial transcriptomics, MERFISH, single-cell, RNA-seq, proteomics, and metabolomics analysis folders.
